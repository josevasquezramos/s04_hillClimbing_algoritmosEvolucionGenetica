Algoritmos de Evolución Genética

¿Qué es un Algoritmo Genético?

Los algoritmos genéticos son métodos de búsqueda y optimización inspirados en el proceso de evolución natural. Se utilizan para encontrar soluciones aproximadas a problemas complejos que son difíciles de resolver mediante otros métodos.

Proceso del Algoritmo Genético

1. Selección

Selección de los mejores individuos de una población para reproducirse.

2. Cruzamiento

Combinación de los genomas de los individuos seleccionados para formar una nueva generación.

3. Mutación

Introducción de pequeñas modificaciones en los individuos para mantener la diversidad genética.

4. Evaluación

Evaluación de la calidad de los individuos con una función de aptitud.

Aplicaciones Comunes

Optimización de Rutas

Problemas de recorrido como el problema del vendedor viajero (TSP).

Redes Neuronales

Diseño de redes neuronales artificiales para clasificación y predicción.

Optimización de Procesos

Optimización de recursos y procesos en industrias y manufactura.

Ejemplo Python: Algoritmo Genético

Usaremos un problema clásico en los algoritmos genéticos llamado "One Max": Se trata de una cadena de 0's y 1's, y el objetivo es encontrar la cadena con la mayor cantidad de 1's. Utilizaremos DEAP para crear la población, la selección, el cruzamiento, la mutación y la evaluación.

Python
¡Copiado! 
                    
import random
from deap import base, creator, tools

# Crear el problema como un individuo con valor de aptitud
creator.create("FitnessMax", base.Fitness, weights=(1.0,))  # Maximizar
creator.create("Individual", list, fitness=creator.FitnessMax)

# Función para crear un individuo (un individuo es una lista de 0's y 1's)
def create_individual():
return [random.randint(0, 1) for _ in range(10)]  # Longitud de 10 bits

# Función de evaluación (suma de los 1's)
def evaluate(individual):
return sum(individual),

# Crear el contenedor de herramientas de DEAP
toolbox = base.Toolbox()
toolbox.register("individual", tools.initIterate, creator.Individual, create_individual)
toolbox.register("population", tools.initRepeat, list, toolbox.individual)
toolbox.register("mate", tools.cxTwoPoint)  # Cruzamiento de dos puntos
toolbox.register("mutate", tools.mutFlipBit, indpb=0.2)  # Mutación de bit con probabilidad de 0.2
toolbox.register("select", tools.selTournament, tournsize=3)  # Selección por torneo
toolbox.register("evaluate", evaluate)

# Crear la población inicial
population = toolbox.population(n=50)  # Tamaño de la población 50

# Parámetros del algoritmo genético
generations = 30
cx_prob = 0.7  # Probabilidad de cruzamiento
mut_prob = 0.2  # Probabilidad de mutación

# Ejecución del algoritmo genético
for gen in range(generations):
# Selección de los mejores individuos (tamaño de torneo 3)
selected_individuals = toolbox.select(population, len(population))

# Crear los hijos a partir de la selección
offspring = list(map(toolbox.clone, selected_individuals))

# Cruzamiento de los hijos
for child1, child2 in zip(offspring[::2], offspring[1::2]):
    if random.random() < cx_prob:
        toolbox.mate(child1, child2)
        del child1.fitness.values
        del child2.fitness.values

# Mutación de los hijos
for mutant in offspring:
    if random.random() < mut_prob:
        toolbox.mutate(mutant)
        del mutant.fitness.values

# Evaluación de los hijos no evaluados
for ind in offspring:
    if not ind.fitness.valid:
        ind.fitness.values = toolbox.evaluate(ind)

# Reemplazar la población actual con los hijos
population[:] = offspring

# Imprimir la mejor solución de cada generación
top_individual = tools.selBest(population, 1)[0]
print(f"Generación {gen}: Mejor individuo {top_individual}, Aptitud: {top_individual.fitness.values[0]}")

# Imprimir el mejor individuo final
best_individual = tools.selBest(population, 1)[0]
print(f"Mejor individuo final: {best_individual}, Aptitud final: {best_individual.fitness.values[0]}")
